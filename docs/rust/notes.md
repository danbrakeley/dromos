## Useful commands

| command                 | description                                                                           |
| ----------------------- | ------------------------------------------------------------------------------------- |
| `cargo clippy`          | linter: covers idiomatic Rust, common gotchas, and performance/complexity suggestions |
| `cargo check`           | fast error checks without full link (maybe like c/c++ compiling w/out linking?)       |
| `cargo doc --open`      | Generates docs and opens it in browser                                                |
| `cargo add <crate>`     | npm install                                                                           |
| `cargo tree`            | npm ls                                                                                |
| `cargo fix`             | Automatically fixes some compiler warnings                                            |
| `cargo clean`           | delete target/ folder                                                                 |
| `cargo fmt`             | format code                                                                           |
| `cargo build`/`run`     | build or build/run code                                                               |
| `cargo build --release` | build release binary                                                                  |
| `cargo test`            | run tests                                                                             |
